# ADRs

Use Architecture Decision Records for durable product or structure choices that future contributors should not rediscover from code.

## Naming

Use zero-padded numbers and a short slug:

- `0001-baseline-review-model.md`
- `0002-key-metrics-are-triage-shortcuts.md`

## Suggested Structure

- Context
- Decision
- Consequences

Keep ADRs short and concrete.
