# Sample Dumps

These files are small, deterministic fixtures for local debugging and future tests.

## Files

- `single-blocked.txt`
  small dump with `RUNNABLE`, `BLOCKED`, and `WAITING`
- `review-baseline.txt`
  baseline-style sample for multi-dump review
- `review-followup.txt`
  follow-up sample where one thread becomes blocked and another changes state

Keep fixtures hand-authored and easy to diff.
